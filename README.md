# Ponto do Grão — Site de Cafeteria

Projeto de estudo desenvolvido a partir do tutorial em vídeo do canal **DevClub | Programação** no YouTube.

## Sobre o projeto

Site institucional para uma cafeteria fictícia chamada **Ponto do Grão**, construído com HTML5 e CSS3 puro, sem frameworks ou bibliotecas JavaScript.

## Seções

- **Hero** — apresentação principal com chamada para ação
- **Sobre nós** — história e diferenciais da cafeteria
- **O que servimos** — cardápio com preços
- **O que dizem de nós** — avaliações de clientes
- **Onde nos encontrar** — mapa e informações de contato

## Tecnologias

- HTML5
- CSS3 (variáveis, Grid, Flexbox, media queries)
- Google Fonts — Roboto
